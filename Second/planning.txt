Routes:

User Authentication ;
     /signup : POST -> create a new user account.
     /login : POST -> Login to the current user account.

Voting :
     /candidates : GET -> Get the list of candidates.
     /vote/:candidates  : POST -> Vote for a specific candidates.

Vote Counts :
     /vote/counts : GET -> Get the list of candidates sorted by their vote counts.

User Profile :
     /profile : GET -> Get the existing User Profile information.  
     /profile/password : Get -> change the user password.


Admin candidates maintain : 
      /candidates : POST -> Create a new candidates.
      /candidates/:candidates : PUT -> Update the existing candidates.
      /candidates/:candidates : DELETE -> Delete the existing candidates.
